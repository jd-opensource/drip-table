## 子表底部渲染函数 subtableFooter

- 描述：子表顶部自定义渲染函数
- 类型：

```typescript
  type SubtableFooter = (
    record: RecordType,
    recordIndex: number,
    tableInfo: DripTableTableInformation<RecordType, ExtraOptions>,
  ) => React.ReactNode;
```

- 默认值：`undefined`
- 说明：[`DripTableTableInformation<RecordType, ExtraOptions>`](/drip-table/types/table-information)
